# TypeScript definitions for [Zeroc ICE](https://github.com/zeroc-ice/ice) [![npm version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url]

## Usage

*tsconfig.json*
```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "ice": ["node_modules/typed-ice"]
    }
  }
}
```
